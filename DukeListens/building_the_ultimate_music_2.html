Building the Ultimate Music 2.0 directory

Jason has started things building the Ultimate Music 2.0 directory at http://music2dot0.wetpaint.com/  It's a wiki, so anyone can jump in.  It seems we need to think a bit about how this is going to be organized ... but it doesn't hurt to be bold and just start adding stuff.  We can figure out how to organize things once we have a bunch of data.  So all of those folks who indicated that they'd be willing to help should head on over to wetpaint and starting contributing.
