Last.fm site update

Last.fm has had a recent site update. They've added a bunch of new features such as playlists,  recommendation radio for all users (not just subscribers), image quilts and more.   My favorite new features  is that the last.fm flash player now updates a user's 'now playing' information. This means that you no longer have to play music with the last.fm client to use the last.fm version of Snapp Radio.  Now you can use the last.fm flash player along with Snapp Radio.
