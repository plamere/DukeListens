Blip.fm API

Blip.fm is often described as a twitter for music. Blip users post ‘blips’ to tracks - and as with Twitter, others can follow your Blips and listen to what you’ve posted. It’s music-micro-blogging. The folks at Blip.fm have built some web services around Blip.fm that let a 3rd party application do all sorts of nifty things with Blip. I've taken a look at the Blip.fm API and have posted a review on the Music Machinery blog. So go check it out.
